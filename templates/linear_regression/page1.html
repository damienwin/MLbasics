Linear Regression Basics

Linear Function
A linear function consists of a y-intercept (b) and weight multiplied to x.
Features
A function can have multiple features, or different independent variables, that affect the outcome y. For example, a housing linear function that determines price may have:
   x1 as the square feet
   x2 as the number of floors
   x3 as the nubmer of rooms.
Weight
Each feature (xi) has a different weight (wi), which is a value that determines the significance of each feature. The example function above can be represented with:
Price = w1x1 + w2x2 + w3x3 + b
Cost
Finally, each function has a cost, or the total sum of the distance from each data point to the line. More of that later on.
