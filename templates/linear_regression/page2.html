What is Linear Regression?

Regression analysis is a method used to model the relationship between variables in a dataset. Its objective is to find the best fit line or function to represent the trend in data points. This line can then predict future outcomes based on given feature values. Let's continue with our house example: We plot data for pricing of houses, where the x value is square footage of the house, and the y value is price.

A line will be created using linear regression. We can then use that line to predict price given square footage.
