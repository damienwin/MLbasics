How do we calculate Cost?

There are many ways to calculate cost. We will be using mean squared error, a common way to measure cost for regression models. Take a look at the function below

  • J represents the cost for the specific w and b values
  • m represents the amount of data points, where each point's cost is summed together, and divided to find the average
  • We find the difference between the prediction line function's y value and y value for each point (i)
