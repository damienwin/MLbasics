About Me

Damien Nguyen is a student at Louisiana State University serving as an Undergraduate Researcher for IntelliSys Lab - an artificial intelligence based research team.

This website serves as an interactive tutorial on linear regression, a fundamental aspect of machine learning. It has been created to showcase Damien's ongoing academic pursuits, for full-stack web design experience, and to help others develop a fundamental understanding of machine learning.

This website consists of HTML, CSS, JS, Plotly, NumPy, and Python in order to create animated plots using regression algorithms and formulas. View the development process here, in the Github Repository.
